SABDAAI - HOW TO RUN GUIDE (STEP-BY-STEP)
=========================================

Follow these steps to start and use the SabdaAI comment moderator, visual dashboards, and browser extension.

STEP 1: ACTIVATE VIRTUAL ENVIRONMENT & INSTALL DEPENDENCIES
-----------------------------------------------------------
1. Open PowerShell or Command Prompt in the project folder ("c:\Users\DELL\Downloads\python testing").
2. Activate the virtual environment:
   * On Windows PowerShell:
     .\.venv\Scripts\Activate.ps1
   * On Windows Command Prompt (cmd):
     .\.venv\Scripts\activate.bat
3. Make sure all dependencies are installed:
   pip install -r requirements.txt


STEP 2: CONFIGURE THE GEMINI API KEY (OPTIONAL)
----------------------------------------------
* SabdaAI runs in "Mock Mode" by default if no API key is set.
* To use real Gemini AI classification:
  1. Open the ".env" file in the project root.
  2. Set your Gemini API key:
     GEMINI_API_KEY=your_actual_api_key_here


STEP 3: RUN THE FASTAPI BACKEND SERVER
--------------------------------------
1. In the activated terminal, run the following command to start the server:
   uvicorn main:app --reload --port 8000
2. Keep this terminal open. The backend is now running at: http://localhost:8000


STEP 4: ACCESS THE WEB DASHBOARD
--------------------------------
1. Open your web browser and go to:
   http://localhost:8000
2. You will see the main SabdaAI Safety Shield interface.
3. Test the features:
   * Single Analysis: Type or paste any Nepali text and click "Analyze Comment".
   * Quick Presets: Click any of the preloaded hateful/offensive/normal Nepali examples.
   * CSV Upload: Click the "CSV Upload" tab, drag and drop a .csv file (where comments are in the first column), and click "Process Batch & Show Charts" to open the interactive dashboard.
   * Feedback Loop: Click "👎 Correct AI prediction" on any card to update its classification on-the-fly and save it to training overrides.


STEP 5: LOAD THE CHROME BROWSER EXTENSION
-----------------------------------------
1. Open Google Chrome.
2. Navigate to: chrome://extensions/
3. Turn on the "Developer mode" switch in the top-right corner.
4. Click the "Load unpacked" button in the top-left corner.
5. Select the "extension" folder located inside your project folder ("c:\Users\DELL\Downloads\python testing\extension").
6. The extension is now active!
   * Click the extension icon in your Chrome toolbar to view the mini status dashboard.
   * Visit websites containing Nepali Devanagari text (e.g. news sites or feeds). The extension will automatically scan the content in real-time as you read/scroll and highlight offensive or hateful sentences.
